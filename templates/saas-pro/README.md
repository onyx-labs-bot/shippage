# ShipPage Template: SaaS Pro

A complete, production-ready SaaS landing page template built with Next.js 14, TypeScript, and Tailwind CSS.

## Features
- Hero with animated gradient background
- Feature grid (6 features)
- Pricing table (3 tiers with toggle for monthly/annual)
- Testimonial carousel
- FAQ accordion
- Email capture
- Fully responsive
- Dark mode
- SEO optimized

## Quick Start
```bash
npx create-next-app my-saas --example https://github.com/onyx-labs-bot/shippage/tree/main/templates/saas-pro
cd my-saas && npm run dev
```

## Customization
1. Edit `app/page.tsx` — all content is in arrays at the top
2. Change colors in `tailwind.config.ts`
3. Replace testimonials with real ones
4. Add your Stripe links to pricing buttons
5. Deploy: `npx vercel --yes`
